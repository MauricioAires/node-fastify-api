## Comandos

```bash
# Iniciar arquivo de configurações do typescript
$ npx tsc --init

# Comando para identificar erros de lint
$ yarn eslint src --ext .ts

# Comando para identificar erros de lint e corrigir
$ yarn eslint src --ext .ts --fix

```
